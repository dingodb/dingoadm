# CurveAdm Plugin

Install plugin
---

```shell
$ dingoadm plugin install PLUGIN-NAME
```

Run plugin
---

```shell
$ dingoadm plugin run PLUGUN-NAME --hosts 'HOST1:HOST2:HOST3' --arg 'ARG1=arg1' --arg 'ARG2=arg2'
```

Remove plugin
---

```shell
$ dingoadm plugin rm PLUGIN-NAME
```

List plugins
---

```shell
$ dingoadm plugin ls
```
